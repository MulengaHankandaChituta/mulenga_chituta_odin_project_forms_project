# Odin Sign-Up Form Project

##  Introduction
This project is part of The Odin Project curriculum. It is a sign-up form for an imaginary service designed to help practice HTML and CSS layout, forms, and styling.

The goal is to recreate a professional-looking registration page using modern CSS techniques like Flexbox and Grid.

---

##  Project Objective
Build a sign-up form that includes:
- A split layout with a background image on the left
- A form section on the right
- Proper form validation styles using CSS pseudo-classes
- Custom fonts and styling

---

##  Technologies Used
- HTML5
- CSS3
- Git & GitHub

---

##  Project Setup

### Step 1: Repository Setup
- Created a Git repository for version control
- Set up basic `index.html` and `styles.css`
- Linked CSS file to HTML
- Added initial placeholder content

### Step 2: Layout Planning
- Reviewed the design reference
- Structured HTML into two main sections:
  - Left side (image + logo)
  - Right side (form)

---

##  Features Implemented

### Layout
- Split-screen design using Flexbox
- Left side displays background image
- Right side contains form content

### Form Structure
- First Name & Last Name side by side
- Email & Phone Number side by side
- Password & Confirm Password side by side
- Submit button placed at the bottom

### Styling
- Custom grid layout for form fields
- White space between inputs using `gap`
- Inputs styled with borders and spacing
- Button styled with custom green color (#596D48)

### Form Validation Styling
- Invalid inputs styled using `:invalid`
- Focused inputs styled using `:focus`

---

##  UI Design Notes
- A semi-transparent dark overlay is used behind the logo to improve readability
- Form is styled for a clean and modern look
- No mobile responsiveness (not required for this project)

---

##  Notes
- JavaScript validation for matching passwords is not included (to be learned later)
- Each input field is validated individually using HTML/CSS only
- Mobile responsiveness is not implemented
